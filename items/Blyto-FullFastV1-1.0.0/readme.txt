add this folder to /server/user/mods
to activate this mod, add the following lines to server.config.json located in /server/user


{
	"name": "FullFastV1",
	"author": "Blyto",
	"version": "1.0.0",
	"enabled": true
}


In order to use the visor/glass you will need to use the round glasses [don't worry they won't show on your character] and attach the visor on the round glasses.
